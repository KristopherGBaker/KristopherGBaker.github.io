Inkwell

The Swift static publishing tool behind this site. A blog generator that grew into a portfolio engine, with content collections, bilingual routing, and its own bundled themes.

SwiftStatic siteToolingOpen source

Inkwell started as a personal blog generator and kept growing until it could drive a whole portfolio. It builds this site: the work case studies, the writing, the bilingual routes under /ja/, the résumé page, and now this Building section.

The thing I like about working on my own generator is that every feature has an immediate, honest test. If a change makes the site worse, I notice the next time I open it.

This section you are reading is the most recent example. The updates below are written with the same tool that renders them.
